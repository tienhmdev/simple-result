# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-07
### Added
- Initial release of `@tienhmjs/simple-result`.
- `Ok`, `Err`, and `Result` types.
- `ok()`, `err()` creator functions.
- `isOk()`, `isErr()` type guards.
- `unwrapOk()`, `unwrapErr()` extraction functions.
